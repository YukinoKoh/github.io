Font Distance

This web app helps you to find similar fonts of your choice, looking at font balances and weight. I hope this project gives you some idea when you choose fonts for a project that involves Japanese writing and English writing. It's on GitHub with MIT License. Further reading ...

Type with Processing

Processing is open source software with which coders can interact with its visual outputs. All types in this project are generated digitally with numbers, angles, or sign/cosign calculation etc. It's on GitHub with MIT License.

Rest... prototype with Framer

This is a prototype of a function that suggests users to rest and reflect own psychological status. An idea behind this prototype is that something to help people rest and reflect a day maybe useful for healthy life. This idea was derived mainly from two previous studies: 'The best interface is No interface’ by Golden Krishna and an app 'Moodnotes' developed by ThrivePort, LLC. Particularly, an element about dark interface to calm down is from the book and recording self psychological pattern is from the app.

Natural light and software

“Light is the most important synchronizing agent for the brain and body,”
said Ivy Cheung, co-lead author and Ph.D. candidate in neuroscience Northwestern University. (Cited from here) Such studies let me think possibility in combining healthcare software and morning light, bright blue sky light, cloudy day light, or evening darkness. While thinking this light/health issue, f.lux became an inspiration to mockup this conceptual work.

Exploring with Google Maps

Which information is indispensable? It is the question asked throughout this little project exploring with Google Maps. Imaging the time people use handwritten map on paper to show recommendation to visitors, I gave hierarchy among information appeared on the map. It's on GitHub with MIT License.

ROT13

ROT13 with Google App Engine is a letter substitution cipher that alters each letter by adding ascii value 13 to the original one.

Blog template for a group of users

It's a web app blog template for a group of users. A design question here was how far users want his/her writing be public. Partially because concept of cloud storage became familiar, privacy need around cloud base writing can be vary from very private personal notes to public writing open to anyone in the world. A design decision for this templates is based on use case rather public use than private and also this code is open to use on GitHub with MIT Licence.

Trailer site with Pyhton

It's a project with Python, configuring a html file that shows title and introduction of movies as well as its youtube trailers. I put some movies that I would recommend to watch. It's on GitHub with MIT Licence.

Property M

Property M is a group of developers, who build houses and community around it. I help them in art direction and creating animated logo that delivers their message that houses are for people who live there and who visit them.

The Urban Grocer’s Guide to Urban Eating

The Urban Grocer's Guide was designed for foodies. We decided its art direction inspired by street arts and looked for mouth-watering photos with Creative Commons license.
